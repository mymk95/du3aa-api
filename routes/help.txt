Du3aaAPI (this API is under development).

Usage:

METHOD  PATH                DESCRIPTION
GET     /api                Shows this page
GET     /api/v2             Returns one random du3aa
GET     /api/v2/du3aa       Returns all du3aa's
POST    /api/v2/du3aa       Creates du3aa
GET     /api/v2/du3aa/:id   Returns a specific du3aa
PUT     /api/v2/du3aa/:id   Updates a specific du3aa
DELETE  /api/v2/du3aa/:id   Deletes a specific du3aa

Params are under development.
